# CJ & Elaine · Student Portfolio with Daily Blog

A simple, modern, and sleek portfolio blog website built as a duo project for Web Development class. Two students, two portfolios, one blog — documenting our daily lives, coding journey, and everything in between.

---

## 🌐 Live Preview
Open (https://github.com/morteius/WebSys-LAB2) in your browser to view the website.

---

## ✨ Features
- **👥 Two separate portfolios** — Each student has their own section
- **📝 Daily blog posts** — New entries added daily
- **🎨 Baby blue theme** — Clean, modern, and easy on the eyes
- **📱 Fully responsive** — Works on desktop, tablet, and mobile
- **🧭 Easy navigation** — Navbar and sidebar for mobile
- **🎯 Individual commits** — Clear Git history showing who contributed what
